# ADVENTURE

This program is a small text-based adventure game

## Usage

You can directly run the pre-compiled binary `adventure.bin`. For more options, look at the [simulator documentation](../../ISS/README.md#usage).

    ./ISS/build/src/zimji -c TEST/adventure/adventure.bin TEST/adventure/data.bin

> **NOTE** The data file is necessary as it contains the text for the game !
